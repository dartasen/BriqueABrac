# vr-project-2019

Brique à brac est un projet reproduisant un casse-brique classique à différents niveaux,
seulement c'est avec la kinect que toutes les intéractions vont se faire !

#Gestes : 
* Restez appuyer 4s avec la main droite sur un bouton pour pouvoir l'enclencher
* Bougez votre haut du corps de gauche à droite pour déplacer la raquette
* Ouvrez votre main droite lors d'une partie pour jeter une balle

Pour la démonstration, voir la vidéo "Presentation".
Pour le jeu compilé, c'est "Brique a brac" dans le dossier "Éxécutable".
Pour le code, soit lancer le projet "Brique à Brac" dans Unity soit aller dans "Brique à Brac\Assets\Scripts"

Les musiques (dont on ne doutera ni de la qualité, ni de la pertinence dans notre jeu) sont dans le dossier "Brique à Brac\Assets\Resources\Musiques", libre à vous de supprimer les mp3 que nous avons mis dans le jeu pour mettre les autres (Unity recréra automatiquement des .meta)

#Vous avez besoin :
* Une kinect V2 que vous branchez sur un port USB 3.0
* Unity version 2018.3.11f1 si vous souhaitez recompiler le code

Bon jeu !
